# 《星临者：神兽誓约》数值设计总结（职业仅定攻击距离）

## 一、英雄属性设计原则

- **职业作用**：仅决定攻击距离，不提供任何基础属性加成。  
- **英雄个性**：每个英雄的 HP、ATK、法力上限、技能效果等完全独立设计。  
- **星级成长**：2星 → HP/ATK ×1.8，3星 → HP/ATK ×3.2（法力、距离不变）。  
- **攻速统一**：所有英雄基础攻速 = 1.0 次/秒（每 60 帧攻击一次）。  
- **移速统一**：所有英雄移动速度 = 每 0.5 秒移动一格。  
- **普攻回蓝**：每次攻击回复 10 法力。

## 五、装备数值（阶段三基础4件）

| 装备     | 效果         |
| -------- | ------------ |
| 铁剑     | ATK +20      |
| 锁子甲   | HP +200      |
| 急速手套 | 攻速 +20%    |
| 蓝水晶   | 最大法力 -15 |

**掉落概率**：每场胜利20%掉落一件。  
**穿戴限制**：1星最多1件，2星及以上最多2件。  
**额外限制**：同一英雄最多装备 **1件蓝水晶**，防止法力叠加速度过快。

## 六、平衡性快速检查（更新后）

- **击杀时间**：同星级战士互打约 10 次攻击死亡（580 / 60 ≈ 9.7）。  
- **羁绊提升**：4种族 + 3职业 约提升整体战斗力 30%（原40%），更可控。  
- **经济节奏**：利息上限降低，鼓励中后期消费；连胜奖励提高，增加翻盘可能。  
- **Boss强度**：阶段四Boss HP 3000，配合弱点机制，需要玩家合理搭配。  
- **技能爆发**：法师单体倍率降低，刺客首次加成减少，防止开场秒后排。
